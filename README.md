## Данные

Объявления листинга с сайта Airbnb, полученные с сайта [Inside Airbnb](https://insideairbnb.com/get-the-data/).

## Структура проекта

Первичную обработку выгруженных с сайта данных производили в ноутбуке:

* `Regression1_Ветров_Александр.ipynb`

На основании первичных данных независимо решали две задачи:

### 1. Задача регрессии


Предсказание процента занятости объекта на ближайшие 30 дней.

Файлы:
* `Regression1_Ветров_Александр.ipynb`
* `Regression2_Ветров_Алексей.ipynb`

### 2. Задача кластеризации

Распределение объектов Airbnb на кластеры.

Файлы:
* `Clusterization1_Ветров_максим.ipynb`
* `Clusterization2_Гузовский_Кирилл.ipynb`
