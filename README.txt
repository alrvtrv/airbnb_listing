### Данные:
Объявления листинга c сайта Airbnb, полученные с сайта

### Структура проекта:
* Первичную обработку выгруженных с сайта данных производили в ноутбуке Regression1_Ветров_Александр.ipynb
* На основании первичиных данных независимо решали 2 задачи:
 * Задачу регрессии - предсказание % занятости объяекта на ближайщие 30 дн. в файлах: 
  * Regression1_Ветров_Александр.ipynb
  * Regression2_Ветров_Алексей.ipynb
 * Задачу кластеризации - распределение объектов Airbnb на кластеры в файлах: 
  * Clusterization1_Ветров_максим.ipynb
  * Clusterization2_Гузовский_Кирилл.ipynb
